Creatures

Core I Studio, Critic: Nikole Bouchard
Fall 2019

Using an illustration of sea monsters by Edward Steed as the starting point for a plan drawing, this project embodies a set of "creatures" through roof framing. The program combines child care with recycling and composting and suggests that architecture should take playfulness seriously.
